# IndigoMovieManager_fork フォルダ構成書

最終更新日: 2026-03-06

## 1. 目的

本書は、`IndigoMovieManager_fork` リポジトリのフォルダ構成を把握しやすくするための一覧資料です。  
開発者が「どこに何があるか」を最短で掴めることを目的に、主要ディレクトリとその役割を整理しています。

## 2. ルート構成（主要ディレクトリ）

```text
IndigoMovieManager_fork/
├─ .agent/                      # エージェント用スキル・補助情報
├─ .cursor/                     # Cursor 用設定/コマンド
├─ .github/                     # GitHub Actions / Issue Template
├─ Converter/                   # 変換処理関連
├─ DB/                          # DB 関連（*.wb 互換運用を含む）
├─ Docs/                        # 設計・仕様・運用ドキュメント
├─ Images/                      # 画像リソース
├─ ModelViews/                  # ModelView 層
├─ Models/                      # ドメイン/データモデル
├─ Properties/                  # AssemblyInfo や設定リソース
├─ Tests/                       # テストプロジェクト
├─ Themes/                      # テーマ/スタイル
├─ Thumbnail/                   # サムネイル旧/既存ロジック群
│  ├─ Adapters/
│  ├─ Decoders/
│  ├─ Engines/
│  └─ Test/
├─ UserControls/                # WPF ユーザーコントロール
├─ Watcher/                     # 監視系機能
├─ scripts/                     # 開発補助スクリプト
├─ src/                         # 分離済み/分離中のサブプロジェクト
│  ├─ IndigoMovieManager.FileIndex.UsnMft/
│  ├─ IndigoMovieManager.Thumbnail.Engine/
│  └─ IndigoMovieManager.Thumbnail.Queue/
└─ tools/                       # 外部ツール・同梱物
   ├─ ffmpeg/
   ├─ ffmpeg-shared/
   └─ sinku/
```

## 3. 主要ファイル（ルート直下）

- `IndigoMovieManager_fork.sln` : ソリューション定義
- `IndigoMovieManager_fork.csproj` : メインプロジェクト定義
- `App.xaml` / `App.xaml.cs` : アプリ起動定義
- `MainWindow.xaml` / `MainWindow.xaml.cs` : メイン画面本体
- `MainWindow.*.cs` : 機能別 partial 実装（MenuActions / Player / Search / Selection / Tag）
- `SettingsWindow.xaml` など : 設定/編集系ウィンドウ
- `README.md` : プロジェクト概要

## 4. 補足

- `Docs/` は初版ドキュメントと日付付きドキュメントが混在しています。更新時は履歴追跡しやすい命名を推奨します。
- `src/` 配下には、将来の責務分離を意識した個別プロジェクトが配置されています。
- `tools/ffmpeg*` は動画関連処理の実行基盤に関係するため、更新時は影響範囲確認を必須とします。
