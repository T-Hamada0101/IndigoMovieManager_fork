# SWF検証フロー図（2026-03-08）

## 図の範囲
- SWF入力
- `FWS` / `CWS` / `ZWS` の署名判定
- 埋め込み画像抽出
- `ffmpeg` 縮退
- サムネイル成功と失敗の分岐

## 図に含めないもの
- `ZWS` のLZMA展開
- `DefineBitsLossless` 系の復元
- `DefineShape` 系の自前レンダリング
- ActionScript 実行を伴う完全再現

```mermaid
flowchart TD
    A["SWFファイルを受け取る"];
    A --> B["署名を確認する"];
    B -->|FWS| C["そのまま解析する"];
    B -->|CWS| D["zlib展開して解析する"];
    B -->|ZWS| E["抽出対象外として失敗扱いにする"];
    C --> F["タグ走査を開始する"];
    D --> F;
    F --> G{"埋め込みJPEGタグがあるか"};
    G -->|Yes| H["DefineBits系から画像を抽出する"];
    G -->|No| I["抽出失敗として記録する"];
    H --> J["代表画像を正規化して保存する"];
    J --> K["captureKindをextractにする"];
    I --> L["ffmpeg候補秒数で1枚取得を試す"];
    L --> M{"画像が取れたか"};
    M -->|Yes| N["白画面判定を行う"];
    M -->|No| O["失敗理由を記録する"];
    N -->|問題なし| P["captureKindをffmpegにする"];
    N -->|白画面寄り| O;
    K --> Q["既存サムネ形式へ整形する"];
    P --> Q;
    O --> R["プレースホルダーへ縮退する"];
    E --> R;
    Q --> S["SWFサムネ成功"];
    R --> T["SWFサムネ失敗または代替成功"];
```
